# Archer Fish

## Overview

Action survival game where fish compete to survive in an arena while avoiding enemy robots and dodging obstacles, using water jet weapons to freeze enemies.

## How to Play

- Fish swim around a 1200x700 pixel arena
- Avoid robots that pursue and try to catch fish
- Dodge obstacles: seaweed, islands, icebergs, coral, magnets
- Shoot water jets to temporarily freeze robots
- Water jets have a 3-second cooldown between shots
- Fish can boost for extra speed
- Survive as long as possible before time runs out
- Fish that get caught by robots are eliminated

## Controls

- **Movement:** Arrow keys or WASD (up/down/left/right)
- **Additional player schemes for multiplayer:** IJKL, Numpad
- **Boost:** Space
- **Shoot:** Mouse/touch to aim and fire water jets

## Scoring

Ranked by survival time. A leaderboard tracks best performances.

## Game Options

- Players: 1-4 (human and/or AI-controlled fish)
- Difficulty: Easy (robots slower), Medium (normal), Hard (robots faster)
  - Easy: 1.5 robot speed
  - Medium: 2.5 robot speed
  - Hard: 3.5 robot speed
- Game duration: 60, 120, 180, 240, or 300 seconds
- Number of enemy robots scales with difficulty

## Features

- Real-time physics with collision detection
- Water jet freeze mechanic
- 5 obstacle types
- AI-controlled fish teammates
- Leaderboard with localStorage persistence
- Pause/resume
- 3 difficulty levels
- Configurable game duration
