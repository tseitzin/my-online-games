# Battle Planes

## Overview

Arcade-style shooting game where the player defends against waves of incoming enemy planes using a lightning bolt weapon.

## How to Play

- Enemy planes fly across the screen from left and right sides
- Aim and fire your lightning weapon to destroy them
- Lightning bolt extends from bottom to top of screen
- Weapon has a recharge period after each shot (must wait for it to recharge)
- Each plane destroyed adds to your score
- Game ends when time runs out
- Try to destroy as many planes as possible

## Controls

- **Mouse movement:** Aim weapon position
- **Mouse click:** Fire lightning bolt

## Scoring

1 point per plane destroyed. The goal is to achieve the maximum score before time expires.

## Game Options

- Plane count: 10, 15, or 20 enemy planes
- Difficulty:
  - Easy: 0.7x plane speed, 1.2x plane size (bigger, slower targets)
  - Medium: 1.0x speed, 1.0x size (normal)
  - Hard: 1.5x plane speed, 0.75x plane size (smaller, faster targets)
- Game duration: Configurable in minutes

## Features

- Lightning bolt weapon with recharge system
- Explosion animations
- Multiple plane varieties (2-4 types based on plane count)
- Difficulty multipliers for speed and size
- Numbered planes for targeting
- Dynamic plane generation
